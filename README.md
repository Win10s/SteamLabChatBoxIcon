# はじめに:

&nbsp;&nbsp;***使用は自己責任でお願いします。***

# どんな機能がありますか
&nbsp;&nbsp;配信者は、名前が<font color="Red">赤色</font>に　モデレーターは名前が<font color="Blue">青色</font>に
&nbsp;&nbsp;VIPは、名前が<font color="Aqua">水色</font>に　サブスクライバーは名前が<font color="Green">緑色</font>になります。
&nbsp;&nbsp;
&nbsp;&nbsp;そして、チャットを打ったユーザーのアイコンが表示されます。
&nbsp;&nbsp;![](https://i.imgur.com/jjwKG4A.png)

# セットアップ
1. CSS,HTML,JS(JavaScript)をダウンロードします。
2. SteamLabのChatBoxからEnable Custom HTML/CSSを有効にします(Enabledにチェックを入れる)。
3. HTMLタブを選択し、表示されたコードを、ダウンロードしたhtmlのコードに置き換えます。
4. CSSタブを選択し、表示されたコードを、ダウンロードしたcssのコードに置き換えます。
5. JSタブを選択し、表示されたコードを、ダウンロードしたjsのコードに置き換えます。
6. Save Settingsを押し、完了です。!
